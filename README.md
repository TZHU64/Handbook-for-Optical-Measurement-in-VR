本手册分为九个章节，全面涵盖了国际标准、测量设备、测量方法以及实际案例研究。书中系统地讲解了如何利用光学测量技术精确测量增强现实（AR）、虚拟现实（VR）和混合现实（MR）设备及其组件。本书旨在帮助感兴趣的人士更深入地理解XR技术，同时规范相关术语和技术知识，为行业发展提供有力支持。
